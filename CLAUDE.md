# このリポジトリについて

このリポジトリは Ayr(Active Youtube Reader)という名前のネイティブアプリを開発するためのものであり、OSS として GitHub にソースコードを公開します。
このアプリはユーザーが受動的ではなく、能動的に Youtube から動画を視聴し、情報を得ることができるような仕組みを提供します。

# 技術スタック

React Native(Expo) + Supabase(Database と Backend)

# コーディングに関して

ユーザー自身が React Native での開発の練習をしたいと考えているので、ある程度ユーザー自身でのコーディングをしていきたい。基本的には質問をメインで聞いていくことになります。プロンプトで与えた機能を Claude に実装して欲しい時は「実装して」という語句が含まれます。
